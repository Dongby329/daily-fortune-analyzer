# 🧧 今日运势分析器

> 结合星座运势 × 中国传统命理，生成你的专属每日运势报告

## 📖 项目简介

今日运势分析器是一款面向个人用户的运势推算工具。用户输入**出生年月日**（必填）和**精确出生时间**（选填推荐），系统结合**西方星座运势**与**中国传统算命**（八字/生肖/五行）进行综合分析，生成一份丰富、合理、有逻辑的每日运势报告。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔢 出生信息输入 | 年月日（必填）+ 精确时间（选填推荐） |
| ♈ 星座运势分析 | 基于 12 星座的今日整体运势计算 |
| 🀄 传统命理推算 | 八字、生肖、五行分析 |
| 📊 综合运势报告 | 整体运势、潜在问题、行动建议、幸运颜色等 |

## 🏗 技术栈

- **前端**：HTML / CSS / JavaScript
- **目标平台**：Web（响应式单页应用）
- **数据**：内置星座对照表、八字基础算法
