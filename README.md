# 🧧 今日运势分析器

> 星座 × 传统命理深度融合，输入出生信息，获得精准、有逻辑的每日运势报告。

---

## ✨ 功能

| 模块 | 说明 |
|------|------|
| 🔢 出生信息采集 | 年月日（必填）+ 精确时间（选填），填写后可解锁八字精算 |
| ♈ 星座运势 | 12 星座判定 + 四维度运势分（事业 / 感情 / 财运 / 健康） |
| 🐉 生肖运势 | 12 生肖运程 + 今日地支关系分析 |
| ☯️ 八字排盘 | 年/月/日/时四柱 + 日主 + 十神分析 |
| 📊 综合评分 | 星座 × 八字加权融合 0-100 分 + 五等级 + 一句话总结 |
| 🧠 性格解析 | 星座性格 × 八字日主交叉分析，双视角画像 |
| 🤝 人际契合 | 今日宜近 / 宜远人群特质动态推荐 |
| 🎨 幸运物 | 幸运色 + 数字 + 方位 + 五行强弱分析 |

---

## 🚀 使用

```bash
# 直接在浏览器打开
open index.html
```

1. 输入你的出生日期（必填）和时间（选填）
2. 点击「查看今日运势」
3. 获得完整的每日运势报告
4. 可分享运势卡片

> 所有计算在浏览器本地完成，数据不上传。

---

## 🏗 技术

| 层 | 技术 | 说明 |
|----|------|------|
| 界面 | HTML + Tailwind CSS CDN | 黑底极简风格，响应式 |
| 星座引擎 | 纯 JS 算法 | 12 星座日期判定 + 元素属性 |
| 八字引擎 | 纯 JS 算法 | 五虎遁 + 五鼠遁 + 节气月柱 + 万年历日柱 |
| 融合引擎 | 加权评分算法 | 有/无出生时间两套权重 |
| 部署 | 纯静态单文件 | 零后端 / 零数据库 / 零依赖 |
